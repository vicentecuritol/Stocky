# Stocky - Sistema de control de inventario para pymes
Proyecto academico desarrollado con **Java Spring Boot** para la asignatura Desarrollo Fullstack I.
## Sobre Stoky
**Stoky** es una app web orientada a pequeñas y medianas empresas para gestionar eficientemente su inventario facilitando procesos como:

- Registro y actualización de productos
- Contol sobre stock
- Consultas sobre historial de movimientos
- Ver proveedores de productos

## Requisitos
- **Java 21** o superior
- **Oracle SQL developer**
- **IDE Recomendado:** Vs Code | IntelliJ IDEA Eclipse 

## Cómo Ejecutar
### 1) Mediante Powershell / CMD de Windows en carpeta raiz de stocky
```bash
    .\mvnw.cmd spring-boot:run
```
